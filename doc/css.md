
- Default size for normal text (like paragraph): 16px (100%,1em,1rem,12pt)                                   
- white-space:  pre-wrap   - Preserve extra spaces and newline ( <br> create also new line )
- display: block - Consumes full width available, with a new line before and after the element
- display: inline - Consumes minimal width. Follow height/width of line . Do not force new lines  
- display: inline-block -  Same as inline but allow to set height/width.  Do not add line-break after

__Examples__ :  
color; green; background-color: red;  font-family: sans-serif; font-weight: bold; font-style: italic; text-decoration: underline; text-align: center;  padding-bottom: 0px ;  margin: 0px;  text-transform: uppercase/lowercase/capitalize;
